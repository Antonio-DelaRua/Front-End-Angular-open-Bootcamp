

- Logica
    - ‘Y’ → 2 o mas cosas deben cumplirse.
    - ‘O’ → 1 o ninguna deben cumplirse

- Comparativas
    - mayor que
    - menor que
    - mayor o igulal que
    - menor o igual que
    - igual a

- codigo 
     - Y -> &&
     - O -> ||
     - mayor que -> >
     - menor que -> <
     - mayor o igual que -> >=
     - menor o igual que -> <=
     - igual a ==
     - distinto a -> !=
